Configurar Rocksdb
==================

Para compilar Rocksdb deberá acceder, a través de la terminal, a la carpeta donde se encuentra la biblioteca Rocksdb:

.. code-block:: shell

	servidor/bibliotecas/rocksdb

Y ejecutar el siguiente comando en la terminal: 

.. code-block:: shell

	make static_lib
