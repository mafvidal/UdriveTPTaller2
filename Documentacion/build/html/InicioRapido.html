Inicio rápido¶

Para arrancar el servidor ejecute los siguientes pasos:

  • Diríjase a la carpeta del proyecto, a través de la terminal.
  • Compilar biblioteca RocksDB:
cd servidor/bibliotecas/rocksdb/
make static_lib
  • Compilar Gtests:
cd ..
cd gmock/
cmake .
make
  • Compilar el servidor:
cd ../..
mkdir build
cd build/
cmake ..
make
  • Arrancar servidor:
./servidor
