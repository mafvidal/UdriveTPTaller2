Información del servidor¶

El servido fue desarrollado en C/C++, en un sistema operativo ubuntu 15.04 de 64 bits, y utilizando el control de versiones GitHub.

Para el servidor HTTP se utilizo la biblioteca Mongoose.

Para el desarrollo de la base de datos se utilizo la biblioteca RocksDB.

Para los pruebas se utilizaron las bibliotecas GMock y GTest.

Como biblioteca para la manipulación de los archivos json, se utilizo Jsoncpp.

La compilación del servidor se realizo mediante CMAKE.
