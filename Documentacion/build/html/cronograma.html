Cronograma¶

Primer checkpoint¶

Para el primer checpoint se desarrollo en el servidor la base de datos, y se testeo su funcionamiento, también se desarrollaron las características básicas de la comunicación del servidor. Para el cliente, se desarrollo una interfaz gráfica, para loquearse y poder ver los archivos disponibles

Segundo checkpoint¶

Para el segundo chekcpoint se desarrollo la REST API en el servidor, y se la testeo, en cuanto a la base de datos, se la modularizo, dividiéndola en clases mas pequeñas.

Tareas a futuro¶

  • Realizar integración servidor-cliente
  • Desarrollar sistema de log para el servidor
  • Completar interfaz gráfica para el cliente
