Información del servidor
========================

El servido fue desarrollado en C/C++, y utilizando el control de versiones `GitHub <https://github.com/mafvidal/UdriveTPTaller2>`_.

Para el servidor HTTP se utilizo la biblioteca `Mongoose <https://code.google.com/p/mongoose/>`_.

Para el desarrollo de la base de datos se utilizo la biblioteca `RocksDB <http://rocksdb.org/>`_.

Para los pruebas se utilizaron las bibliotecas `GMock y GTest <https://code.google.com/p/googlemock/>`_.

Como biblioteca para la manipulación de los archivos json, se utilizo `Jsoncpp <https://github.com/open-source-parsers/jsoncpp>`_.

La compilación del servidor se realizo mediante `CMAKE <https://cmake.org/>`_.
