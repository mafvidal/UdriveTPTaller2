.. _limitaciones:

Limitaciones
============

El servidor cuenta con las siguientes limitaciones:

* La información guardada por la base de datos se almacena en el directorio **datosdb**.
* Los archivos que envía el cliente se almacenan en el directorio **Udrive**.
* El servidor siempre utiliza el puerto 8000 para las comunicaciones con los clientes.
* Siempre al ejecutar el servidor se crea el archivo **Log.txt**, y al volver a ejecutar el servidor, el log se agrega al archivo existente.
* Por cada vez que se corren los tests se debe eliminar la carpeta **datosdb**.
