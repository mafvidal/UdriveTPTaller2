Integración continua
====================

La integración del proyecto se realizo mediante el servidor Tavis CI, para ello se vinculo la `cuenta del proyecto en GitHub <https://github.com/mafvidal/UdriveTPTaller2>`_ con la `cuenta en Travis CI <https://travis-ci.org/mafvidal/UdriveTPTaller2/builds>`_.

Luego se configuro el archivo .travis.yml, para que luego de cada commit, Travis CI, ejecute los test del proyecto, y envíe un mail a los integrantes del grupo con el resultado de la corrida de los tests.
